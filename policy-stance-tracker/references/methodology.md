# 方法论细则

## 1. 为什么用"交易主义强度"量表，而不是简单的"友好/敌对"

传统的立场追踪常用"友好—中立—敌对"三分法，但这套框架不适合分析"交易主义外交风格"这个具体研究问题——
特朗普对日本的表态经常在"盟友语言"和"要价语言"之间快速切换，而不是单纯变友好或变敌对。
所以本skill默认量表衡量的是**表态在多大程度上把安全/外交承诺和具体经济对价挂钩**，而不是笼统的情感倾向。

如果你研究的不是交易主义风格，而是别的机制（比如意识形态输出、国内政治周期影响外交），
请把下面的量表换成你自己的编码标准，但仍然保留"每个分值必须有原文依据"这条规则。

## 2. 交易主义强度量表（默认）

| 分值 | 判定标准 | 常见语言标志 |
|---|---|---|
| 0 | 表态围绕共同价值观、历史友谊、制度性承诺，不提具体对价 | "基石"、"共同价值观"、"长期伙伴" |
| 1 | 提及互惠但仍以联盟框架为主，没有具体金额/条款 | "公平互惠"、泛泛提及"该做更多" |
| 2 | 明确提出具体让步要求（关税税率、市场准入品类等）作为条件 | 具体商品清单、具体关税数字 |
| 3 | 提出具体金额/份额的单边要价，或把安全承诺明确与经济让步挂钩 | 具体驻军费用数字、"如果不…就…"句式 |

打分时注意：**行动（如签署撤出某协议的行政令）本身也可以打分**，不是只有口头表态才算数——
撤出多边框架、转向双边谈判，这个行为本身就是交易主义强度的证据，要在 `content_paraphrase` 里说明这是"行动"还是"言论"。

## 3. 语境标注模板

```
occasion: [峰会联合声明 / 国内演讲 / 记者会即兴发言 / 被曝光的内部讨论 / 谈判信函 ...]
audience: [国内选民 / 谈判对手政府 / 本国官僚系统 / 国际媒体 ...]
surrounding_events: [前后1-2周内的相关事件，尤其是可能改变说话动机的事件]
possible_misreading: [这条表态单独看容易被误解成什么，为什么]
```

**"被曝光的内部讨论"要特别小心**：比如通过回忆录、爆料披露的表态（如某位前官员书中提到的内部要价），
它反映的可能是谈判中的极端开价策略，不完全等同于对外公开宣示的政策立场。这类条目在 Step5 人工审计时必须
额外注明信源类型（"回忆录转述" vs "官方文件"），供人工判断权重。

## 4. 休谟三原则检查清单（用于Step 5的"排除混淆"）

判断一个疑似的"立场转变"是否成立，逐项检查：

1. **时间顺序**：这条新表态确实晚于上一个节点，且中间没有被你遗漏的中间态？
2. **共变关系**：立场分值的变化和你假设的自变量（比如"换了对日团队负责人"或"某次国内选举"）在时间上是否对得上？
3. **排除混淆**：这个变化会不会单纯是因为**场合不同**（比如从竞选集会发言变成正式峰会声明，语言天然会更克制），
   而不是实质立场真的变了？如果是，这不算一个真实的"转变"，只是"语域切换"。

三项都过了，才能在最终时间线里标记为"确认的立场转变"；有一项没过，标记为"存疑，需要更多证据"。
